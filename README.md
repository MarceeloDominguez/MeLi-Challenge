## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/MeLi-Challenge.git
```

Go to the project directory

```bash
  cd MeLi-Challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
